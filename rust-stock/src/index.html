设置

磨砂玻璃质感：半透明 + 模糊 + 高光描边 + 浮动药丸 Tab。仿 iOS「液态玻璃」的 WebView 近似实现，夜/日皆适配；极老旧安卓机型会自动降级为不透明底色。
涨跌幅阈值（%）
开启后，自选股当日涨跌幅达到阈值即弹出提醒（仅交易日，每只股 5 分钟内只提醒一次）。纯本地判断，不上传任何数据。
✓ 已接入 API Key
高级：Base URL / 模型（默认 DeepSeek，一般不用动）
所有设置只存本机 SQLite。填入 key 后启用：自选股 AI 打分、情绪 AI 解读、今日推荐、AI 聊天。Base URL/模型可切换任意 OpenAI 兼容服务（Kimi、通义、本地 Ollama 等）。
