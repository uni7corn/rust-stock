# rust-stock VIP · 版本说明与更新日志

> 本文档为 **rust-stock VIP 版** 的功能说明与更新记录。开源版（本仓库主体）永远免费、纯本地、需自备 AI Key；VIP 版面向不想折腾 Key、希望云端集成与更省心体验的用户。

## 开源版 vs VIP 版

| | 开源版（GitHub） | VIP 版 |
|---|---|---|
| 价格 | 永久免费 | 订阅制 |
| AI Key | 需自己申请并填入 | **云端统一集成，无需任何 Key** |
| 账户 | 无需登录，纯本地 | 账户登录（用户名 + 密码 + 邮箱找回） |
| AI 能力 | 自费、自管 | 开箱即用，登录即可用全部 AI 功能 |
| 数据存储 | 纯本地 SQLite | 行情/自选/调研等仍**纯本地**；仅账户与订阅状态在云端 |
| 在线更新 | 手动下载 | 规划中（应用内检测新版） |
| 平台 | Windows / Android（macOS 可自行编译） | Windows / Android |

> 两个版本可在同一台设备**并存安装**（标识不同）。VIP 版的客户端界面风格与开源版一致；只有官网/落地/后台等 Web 端采用 Nothing Design 风格。

## VIP 功能概览

- **登录账户**：用户名 + 密码注册，邮箱用于找回密码；登录后按订阅有效期开通全部 AI 功能。
- **云端 AI，零配置**：AI 分析、AI 聊天、深度调研、K 线预测全部由云端统一提供，**无需也无法自填 API Key**——省心、安全。
- **订阅管理**：到期自动停用 AI 能力并提示续费；续费后立即恢复。
- **AI 预测胜率**（本地）：每次点「AI 预测」自动记录，按"基准日之后第一个交易日收盘价"自动结算涨跌方向是否符合 AI 预测，给出净胜率（可为负）；可在设置页「查看预测记录与结果」翻页回看每一笔的预测内容与实际结果。
- **人气榜增强**：人气榜显示实时现价，并支持一键加入自选。
- **更小的安装包**：安卓仅打包 arm64，安装包显著减小。

### 2026-06-20 · 量价标签：数据时间 + 每分钟刷新 + 永久留存
- **永久保留最近一份**：自选名称右边的量价标签会一直显示最近一次成功获取的信号，**不再显示「无数据」**、不再空白。
- **告知数据时间**：标签后面标注这份数据来自何时——**当天的显示到分钟**（如 `14:35`），**非当天的显示年月日**（如 `2026-06-20`），一眼知道是不是最新。
- **每分钟自动刷新**：标签每分钟自动更新一次，不分当前在哪个页面，**后台也保持刷新**（PC 最小化到托盘持续刷新；手机深后台被系统冻结时，回到前台立即补刷）。有新数据即覆盖为最新。
- 排版自适应：标签与时间字号略缩小以容纳。

### 2026-06-20 · 量价标签休市也能显示（日K兜底）
- 此前量价标签只依赖盘中实时量比接口，**休市/盘后该接口无数据时名称右边会空白**。现改为：以**日K**为底计算换手率/量比/位置/趋势（任何时候都有），**盘中有实时数据则覆盖**（更准）。休市、盘后照常显示上一交易日的量价信号，不再空白；实在没有数据（新股/停牌）才显示「无数据」。

### 2026-06-19 · 量价「四象限实战模型」+ 个股实战提示
- 把《量比+换手率 实战用法》的四象限模型融入核心算法，并结合**价格位置**（低位/高位）与**趋势**判定：
  - **双高（量比>2 且 换手>5%）+ 低位** = 主力吸筹/拉升 → 最强买入（放量突破/站稳均线更强）；处高位则「警惕见顶」。
  - **量比>3 但换手<3%** = 冷门试盘/一日游 → 别追高，等换手站上 5%。
  - **换手>10% 且量比<1** = 维持性放量、动力枯竭 = 出货（危险）。
  - **量比<1 且换手<2%** = 缩量盘整 → 观望；上升通道里=洗盘。
  - 另含：动力不足、温和/明显放量、高位巨量派发（死亡换手）等。
- **个股 K 线页新增「量价实战提示」面板**：显示当前标签 + 实战逻辑 + 操作建议 + 位置/趋势/主力资金上下文；自选股标签长按（悬停）也能看到提示。
- 客户端与服务端判定严格一致；今日推荐的 AI 规则同步为四象限口径（结合位置、趋势、资金流向）。

### 2026-06-19 · 量价信号全覆盖（每只票都有标签）
- 之前换手率≤5%、或落在组合之间的票不显示标签，容易让人以为出 bug。现在把「换手率 × 量比」补成完整矩阵，**每只自选股都有明确状态**：
  - **低换手区（换手≤5%）**：量比<2 `不活跃`、2~3 `略微启动`、3~5 `启动中`、>5 `高额启动中`（仅作量能提示，不影响推荐）。
  - **活跃区（换手>5%）**：`动力不足`（量比<2）/ `启动前兆`（2~3）/ `开始爬升`（换手5~10%且量比3~5，强力推荐）。
  - **放量过度 = `出货·危险`（红字）**：换手 5~10% 且量比>5，或 换手>10% 且量比>3——主力借量出货风险大，坚决不推荐。
  - **停牌 / 拿不到数据** → `无数据`。
- 客户端与服务端判定严格一致；推荐侧的 AI 规则同步更新为上述出货口径。

### 2026-06-19 · 今日推荐实时进度条
- **AI 生成今日推荐时显示进度条**，让你清楚看到「梳理推荐股」进行到哪一步、还是不是卡住了：
  - **准备候选池**（4%）→ **候选池就绪、AI 初选**（15%）→ **初选完成、逐股深度调研**（40%）→ 每完成一支深调研进度继续推进（40%→98%）→ **完成（100%）自动展示结果**。
  - 进度由后端真实阶段驱动；初选这段较慢时进度条仍会缓慢前移，确认程序在运行而非卡死。
  - 文案实时显示当前阶段（如「深度调研 3/7 支…」）。

### 2026-06-19 · 多设备实时同步 + 切页即刷
- **多设备自动同步**：在一台设备（如 PC）增删自选/调研/预测后，另一台设备（如手机）**无需退出重进**即可自动同步：
  - **切到自选页**先与云端对账，立即拉到刚加的股票；
  - **回到前台**（手机从后台切回）自动对账补拉；
  - **后台周期对账**（每 45 秒）：PC 最小化到托盘仍持续保持更新；手机深后台被系统冻结时，回前台那一刻立即补拉。
- **对账策略**：云端较新→下拉并重渲染；本机较新→补推；相等→不动（按时间戳 last-write-wins，避免互相覆盖）。
- **切页即刷新**：每次切换页面都对该页做一次网络刷新（行情/情绪/板块/人气榜/自选/快讯/会员到期），不必等下一拍定时器，数据始终是切进来那一刻的最新。

### 2026-06-19 · 核心量价算法（换手率 × 量比）
- **新增一条核心算法规则**，客户端与服务端**双重严格判定**，对每只推荐个股与每只自选个股都判定其「换手率 × 量比」组合：
  - **换手 > 5%，量比 2~3** → 资金悄悄进场、动作很轻，是**启动前兆**：纳入推荐；自选显示 `[启动前兆]`。
  - **换手 5%~10%，量比 3~5** → 量价齐升、强势启动，是**开始爬升**：**强力推荐、给高分**；自选显示 `[开始爬升]`。
  - **换手 > 10%，量比 > 5** → 看价格位置：**低位 = 主力大力吸筹**（推荐，自选显示 `[主力已经开始吸筹]`）；**高位 = 借量出货**（**坚决不推荐**，自选红字强提示 `[主力出货，注意危险]`）。
  - **换手 > 5%，量比 < 2** → 热度还在但没人接盘、主力大概率已撤场，是**动力不足**：**不推荐**；自选显示 `[动力不足]`。
- **推荐侧**：候选池预筛阶段直接剔除「动力不足」，并按形态加分；AI 推荐提示词强制注入上述四组合规则，命中高位出货的一律剔除/降分并在理由里点明。
- **自选侧**：每只自选股实时拉取换手率/量比并打标签；仅「放量剧烈需看位置」时才补拉日K判定高/低位，最省流量。

### 2026-06-15 · 推荐双标签 + 自填 Key + 设置分类
- **今日 AI 推荐分两个标签**（像人气榜分人气/资金流入）：「**热点股**」用原算法挖今日活跃热门（涨幅/主力净流入/龙虎榜＋多流派），「**潜力股**」用新两段式挖低估值中小盘。一键切换、各自独立历史与★连推。
- **「潜力股」两段式**：先本地规则预筛（中小市值、估值不高、当日未大涨、量能温和，排除已涨停/高估），再 AI 按「数倍逻辑（行业空间×渗透/份额拐点×估值修复）+ 多流派裁决 + 价值陷阱排查」深度推理——挖被低估、还没启动的潜力股，而不是追已涨停的热门。点单支仍可做产业链深度调研。
- **可自填 AI 接口**：在「设置 → AI」填入自己的 DeepSeek/OpenAI 兼容 Key，AI 将**直连你的接口**（不经我们服务器，流量高峰更快更稳）；不填则继续用云端。算法始终由我们提供。
- **设置改为分类折叠**：行情数据源 / 主题 / AI / 系统 四个折叠，默认收起、点击展开；个人信息置顶常显。

### 2026-06-14 · AI 预测引擎升级（随下个版本发布）
