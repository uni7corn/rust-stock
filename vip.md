# rust-stock VIP · 版本说明与更新日志

> 本文档为 **rust-stock VIP 版** 的功能说明与更新记录。开源版（本仓库主体）永远免费、纯本地、需自备 AI Key；VIP 版面向不想折腾 Key、希望云端集成与更省心体验的用户。
>
> 倒序排列，最新更新在最上方。每次 VIP 内容更新都会同步追加到这里。

---

## 开源版 vs VIP 版

| | 开源版（GitHub） | VIP 版 |
|---|---|---|
| 价格 | 永久免费 | 订阅制 |
| AI Key | 需自己申请并填入 | **云端统一集成，无需任何 Key** |
| 账户 | 无需登录，纯本地 | 账户登录（用户名 + 密码 + 邮箱找回） |
| AI 能力 | 自费、自管 | 开箱即用，登录即可用全部 AI 功能 |
| 数据存储 | 纯本地 SQLite | 行情/自选/调研等仍**纯本地**；仅账户与订阅状态在云端 |
| 在线更新 | 手动下载 | 规划中（应用内检测新版） |
| 平台 | Windows / Android（macOS 可自行编译） | Windows / Android |

> 两个版本可在同一台设备**并存安装**（标识不同）。VIP 版的客户端界面风格与开源版一致；只有官网/落地/后台等 Web 端采用 Nothing Design 风格。

---

## VIP 功能概览

- **登录账户**：用户名 + 密码注册，邮箱用于找回密码；登录后按订阅有效期开通全部 AI 功能。
- **云端 AI，零配置**：AI 分析、AI 聊天、深度调研、K 线预测全部由云端统一提供，**无需也无法自填 API Key**——省心、安全。
- **订阅管理**：到期自动停用 AI 能力并提示续费；续费后立即恢复。
- **AI 预测胜率**（本地）：每次点「AI 预测」自动记录，按"基准日之后第一个交易日收盘价"自动结算涨跌方向是否符合 AI 预测，给出净胜率（可为负）；可在设置页「查看预测记录与结果」翻页回看每一笔的预测内容与实际结果。
- **人气榜增强**：人气榜显示实时现价，并支持一键加入自选。
- **更小的安装包**：安卓仅打包 arm64，安装包显著减小。

## 更新日志

### 2026-06-14 · v1.7.0
- **在线自动更新**：后台上传新版安装包并填写版本号后，客户端启动时自动检测；发现新版弹卡片，一键更新——不必再到官网手动下载。
  - Windows：应用内下载安装包（带进度条）+ SHA-256 完整性校验 → 自动启动安装程序并退出本体覆盖安装。
  - Android：一键打开下载（系统下载器接管），下载完成点一下即可安装（系统限制，App 无法静默装 APK）。
  - 后台「应用发布」新增**版本号**输入：版本号需大于线上版本，客户端才会提示更新。

### 2026-06-14 · v1.6.0
- **个股估值上屏**：K 线详情新增 PE-TTM / PE(静) / PB / 总市值 / 流通市值 / 换手率（腾讯行情）。
- **机构研报 + 一致预期**：聚合东方财富券商研报，计算近 15 篇的一致预期 EPS（今年/明年/后年）与评级分布，详情页展示最新 5 篇研报。
- **喂给 AI 预测**：估值指标与机构一致预期 EPS 现已注入「AI 预测」与「AI 分析」的上下文——基本面锚点入模，持续提升 K 线涨跌预测准度。
- 数据接口字典参考开源项目 `simonlin1212/a-stock-data`（Apache-2.0），未复制其代码，见仓库 NOTICE。

### 2026-06-13
- **AI 预测记录页**：新增翻页（每页 10 条）、记录上限 200（超出自动清理最旧）、一键清空。
- **人气榜**：显示实时现价 + 一键加入自选。
- **修复**：「问问 AI」与「深入调研」此前在 VIP 版仍被要求填 Key，现已改为按登录与订阅状态判定，登录即用。
- **图标修复**：安卓启动/安装图标的独角兽脸部黑块修正为正常白色。
- **安卓瘦身**：安装包默认仅含 arm64 架构，体积由 40MB+ 降至约 13MB。
- **设置页**：移除与 VIP 无关的旧版「自填 Key」相关说明。
- **下载托管**：官网提供 Windows 与 Android 安装包下载。

### 2026-06-13（VIP 架构首发）
- **账户体系上线**：注册 / 登录 / 邮箱找回密码；登录门在 App 启动时拉起。
- **AI 全面云端化**：彻底移除客户端自填 API Key 的入口与代码，所有 AI 调用经云端统一提供。
- **订阅校验**：服务端按订阅到期时间放行 AI；到期拒绝并提示续费。
- **AI 预测胜率**：本地记录每次预测与自动结算的胜率统计。
- **Web 端品牌升级**：官网、找回密码页、后台管理台采用 Nothing Design 暗色风格，统一使用独角兽品牌 LOGO。
